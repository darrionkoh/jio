# Jio!

A clean and simple going-out toolkit built for Singaporeans, by Singaporean. Built with HTML, CSS, and JavaScript.

## Features

- **Bill Splitter** — add people, assign items, auto-splits with GST and service charge
- **GST Calculator** — quickly calculate Singapore's 9% GST and 10% service charge
- **Owe Money Pay Money** — track who owes who across multiple outings
- **Food Finder** — find nearby hawker centres, restaurants, or cafes using your location


## Singapore Tax Rates

- GST: 9% (applies on subtotal + service charge)
- Service Charge: 10% (applies on subtotal)
